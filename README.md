## 🚨 The Problem
Traditional financial security methods (passwords, 2-factor authentication, face recognition) are becoming less effective against Agentic AI. Advanced bots can now mimic human speech, typing patterns, and navigation to completely bypass standard biometric checks. The fatal flaw in modern banking is that **current systems react only after funds are transferred**.

## 💡 Our Solution
Checking *what* a user types is no longer secure. We must verify **how** they interact. 

**BOTGUARD** is a proactive, front-end security platform powered by our proprietary **Neuro-Shield Engine**. Instead of waiting for a fraudulent transfer to complete, our engine monitors micro-behaviors in real-time. If Agentic AI behavior is detected, BOTGUARD acts as an instant killswitch—terminating the session before a single rupee leaves the account.

## 🎯 Core Features (The Behavioral Traps)

Our High-Fidelity Web MVP showcases 5 real-time traps designed to catch automated scripts:

1. **Typing Biometrics (Superhuman Speed):** Analyzes the millisecond delay between keystrokes. Instantly flags credential-stuffing bots that paste or inject data at 0ms speeds.
2. **Invisible DOM Honeypots:** Bots scrape HTML to find input fields. We embedded invisible input fields (e.g., a hidden Phone Number box) that humans cannot see, but bots will eagerly auto-fill.
3. **Adaptive UI Traps (Spatial Wiggle):** Bots click hardcoded X/Y coordinates. Our critical buttons slightly shift position upon hover. A human follows the button; a bot clicks the empty dead space.
4. **Cursor Trajectory Analysis:** Calculates the mathematical linearity of mouse movements. Bots move in 100% straight lines; humans have natural, biological curves.
5. **Hardware Gyroscope Simulation:** Verifies natural human hand-shake and device tilt, ensuring the interaction is happening on a real physical device.
